# COTATO AWARDS

8TH COKERTHON 수상 팀을 3등부터 1등까지 순차적으로 공개하는 행사용 프레젠테이션 화면입니다.

## 수상 정보 설정

대기 화면 우측 상단의 `AWARD SETUP` 버튼에서 1·2·3등 팀명과 상금을 미리 저장합니다. 기본 상금은 1등 15만 원, 2등 10만 원, 3등 5만 원입니다. 설정값은 현재 브라우저에 저장됩니다.

## 조작

- `Space`, `Enter`, `ArrowRight`: 다음 단계
- `ArrowLeft`: 이전 단계
- `R` 0.9초 누르기: 처음으로 초기화
- 우측 상단 버튼: 전체화면 전환

```powershell
corepack pnpm --filter @cotato/hackathon-awards dev
```

단독 실행 주소는 `http://127.0.0.1:4180`입니다.
