# 행운의 감자 추첨

1부터 41까지의 번호 중 원하는 개수만큼 중복 없이 추첨하고, 감자 캐릭터가 핀볼 보드를 통과하며 결과를 공개하는 임준서 담당 기능입니다.

```powershell
pnpm --filter @cotato/lucky-potato-draw dev
```

단독 실행 주소는 `http://127.0.0.1:4175`입니다. 포털에서는 `/features/lucky-potato-draw`로 접근합니다.

## 추첨 원칙

- 추첨 범위는 1부터 41까지입니다.
- 선택 가능한 개수는 1개부터 41개까지입니다.
- 쉼표나 띄어쓰기로 특정 번호를 입력하면 다음 레이스 참가자에서 제외할 수 있습니다.
- Web Crypto 난수로 번호를 먼저 확정한 뒤 핀볼 애니메이션으로 순서대로 공개합니다.
- 한 번의 추첨 안에서는 같은 번호가 두 번 나오지 않습니다.
- 당첨 감자가 먼저 골인한 뒤 나머지 참가 감자도 약 15초 안에 모두 완주합니다.
- 라운드마다 감자별 가속·감속 파동이 달라져 실시간 선두와 TOP 3가 계속 바뀝니다.
- 출발과 함께 캐주얼 칩튠 BGM이 재생되며 화면 상단에서 켜고 끌 수 있습니다.
- 레이스가 끝나면 당첨 감자와 순위를 큰 결과 화면으로 보여줍니다.
- 전체화면 버튼은 브라우저 Fullscreen API와 화면 채우기 폴백을 함께 지원합니다.
