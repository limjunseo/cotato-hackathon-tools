# 행운의 감자 추첨

1부터 41까지의 번호 중 원하는 개수만큼 중복 없이 추첨하고, 감자 캐릭터가 핀볼 보드를 통과하며 결과를 공개하는 임준서 담당 기능입니다.

```powershell
pnpm --filter @cotato/lucky-potato-draw dev
```

단독 실행 주소는 `http://127.0.0.1:4175`입니다. 포털에서는 `/features/lucky-potato-draw`로 접근합니다.

## 추첨 원칙

- 추첨 범위는 1부터 41까지입니다.
- 선택 가능한 개수는 1개부터 41개까지입니다.
- Web Crypto 난수로 번호를 먼저 확정한 뒤 핀볼 애니메이션으로 순서대로 공개합니다.
- 한 번의 추첨 안에서는 같은 번호가 두 번 나오지 않습니다.
